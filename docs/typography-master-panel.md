# Typography Master Panel

The Typography Master appears as a wide docked panel beside the live menu preview. It uses an off-white background, soft borders, rounded sections and generous spacing so it feels approachable rather than technical.

The panel is divided into four numbered sections.

## 1. Global controls

The first section contains a small number of controls affecting the whole menu:

- **Font set** - selects a coordinated font pairing, such as Playfair and Inter
- **Overall text size** - scales the menu typography proportionally
- **Density** - switches between Compact, Balanced and Spacious layouts

These controls allow a user to make broad visual changes without editing every typography role separately.

## 2. Typography roles

The second section lists the main types of text used in the menu:

- Menu title
- Section header
- Dish name
- Dish price
- Dish description
- Dietary key
- Footer

Each row includes:

- A simple icon
- The role name
- A small preview of the current style
- An arrow to open its settings

The selected role is softly highlighted.

Selecting text in the menu preview should automatically select the corresponding role in this list.

## 3. Selected role

The third section shows controls for the selected role.

For Dish Name, the panel includes:

- Font
- Size
- Weight
- Alignment
- Capitalisation
- Space above
- Space below

Controls are arranged clearly in short rows, using dropdowns, button groups and sliders.

Changes update every linked item in the live preview immediately.

## 4. Advanced settings

Advanced settings remain collapsed at the bottom of the panel.

Opening this section may reveal:

- Exact line spacing
- Letter spacing
- Automatic fitting
- Minimum and maximum sizes
- Wrapping behaviour
- Print readability controls

These settings are available when needed but remain hidden during normal use.

## Panel behaviour

The panel can be:

- Docked beside the menu preview
- Resized
- Collapsed
- Combined with other panels as a tab
- Detached as a floating panel

The menu preview remains visible while typography is edited, allowing the user to see changes instantly.

The overall design should feel calm, spacious and guided. It should present the most useful controls first and keep technical complexity out of sight.
