Griffin Menu Studio 1.1.1 — Tidier update prompts

A small fix to the update experience.

## Fixes

- **Cleaner update card.** The Updates card in Settings now shows just **Update Now** and **Later** — the buttons no longer overlap in a narrow card, and the redundant "Skip this update" control has been removed.
- The download notification is likewise simplified to **Update Now** and **Later**.

Thank you for using Griffin Menu Studio.
