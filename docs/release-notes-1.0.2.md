Griffin Menu Studio 1.0.2 — Tidier Settings

A small update that streamlines the Settings page.

## What's new

- **One Storage card.** Backup & recovery is no longer a separate card — its autosave controls now sit inside the **Storage** card, alongside your folder locations, so everything about where your work lives is in one place.
- **Renamed.** "Storage locations" is now simply **Storage**.

Thank you for using Griffin Menu Studio.
